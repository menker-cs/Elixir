# Contributing To The Menu
If you happen to find a bug with the menu, create an **[issue](https://github.com/menker-cs/elixir/issues)** and we'll be quick to fix it ASAP!
If you want to contribute to the menu, or an fix an issue yourself make a **[pull request](https://github.com/menker-cs/elixir/pulls)**. 
Depending on how critical of an issue your pull request fixes you may be given a role in our Discord server to mark the importance of your contribution
