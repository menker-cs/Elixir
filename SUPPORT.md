# Support

Thanks for using **Elixir**! Your support keeps this project going.

Before opening an issue, check our **[Discord](https://discord.gg/QFeUpmg8vd)**.

## Have questions?
- Post in **#help** or create a ticket.
- Follow [No Hello](https://nohello.net/) & [Don't Ask To Ask](https://dontasktoask.com/).

## Found a bug?
- Use **Issues** (after checking Discord) or a Discord ticket.
- Include: what you were doing, mods/plugins used, error logs/version.

## Feature requests?
- Post in **#suggestions** on Discord.
- Explain how it works + inspiration.
- Large ones take time (solo dev here).

## Response times
- Maintained solo by Menker, so updates vary.
- Repeated asks may be ignored—patience appreciated!

## Help out?
- Join Discord and stay active.
- Helpful users may become staff.
